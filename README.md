# Lab4_Copilot

Git ローカルリポジトリ操作をテーマにした学習用ワークスペースです。  
Markdown ドキュメントと GitHub Copilot を活用した Git 操作マニュアルが含まれています。

---

## ファイル構成

| ファイル | 説明 |
|---|---|
| [gitmanual.md](gitmanual.md) | Git ローカルリポジトリ操作マニュアルの**総合目次**。基本ワークフローの図解と各セクションへのリンクをまとめたエントリーポイント。 |
| [section1.md](section1.md) | **第1章 リポジトリの初期化** — `git init` を使ってローカルリポジトリを作成する手順を解説。 |
| [section2.md](section2.md) | **第2章 ファイルのステージング** — `git add` でファイルをステージングエリアに登録する方法を解説。 |
| [section3.md](section3.md) | **第3章 コミット** — `git commit` でステージング済みの変更をリポジトリに保存する手順を解説。 |
| [section4.md](section4.md) | **第4章 ブランチの操作** — `git branch` / `git switch` を使ったブランチの作成・切り替えを解説。 |
| [section5.md](section5.md) | **第5章 マージ** — 別ブランチの変更を現在のブランチに統合する `git merge` の手順を解説。 |
| [section6.md](section6.md) | **第6章 状態・差分の確認** — `git status` / `git diff` で作業ディレクトリの状態を確認する方法を解説。 |
| [section7.md](section7.md) | **第7章 ログの確認** — `git log` でコミット履歴を参照・調査する方法を解説。 |
| [section8.md](section8.md) | **第8章 変更の取り消し** — 作業中・ステージ済み・コミット済みの各段階で変更を元に戻すコマンドを解説。 |
